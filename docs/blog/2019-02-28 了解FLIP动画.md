## FLIP是什么

FLIP表示First, Last, Invert, Play。即设定元素的开始状态后，再给元素添加结束状态，再利用transform，opacity等Invert掉结束状态，动画开始时去掉这个Invert，这样

First: 元素过渡的初始状态
Last: 元素过渡完成的状态
Invert: 假设一个元素First top: 0px，Last top: 90px，那么需要使用transform: translateY(-90px)，将其
